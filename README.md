## ✨ Features

- **3×3 Game Board** — Clean, responsive grid with smooth animations
- **Two-Player System** — Alternating turns between Player X and Player O
- **Winner Detection** — Highlights the winning line with glowing effects
- **Draw Detection** — Displays a draw message when the board is full
- **Score Tracking** — Persistent scoreboard across multiple rounds
- **Reset Options** — New Round (keeps scores) and Reset All (clears everything)
- **Confetti Celebration** — Falling confetti animation on winning
- **Premium Dark UI** — Glassmorphism design with animated gradients

## 🛠️ Tech Stack

- **React 19** — Component-based UI with hooks
- **Vite** — Lightning-fast build tool
- **Vanilla CSS** — Custom design tokens, animations & glassmorphism

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Sahilprashar02/Tic-Tac-Toe-Game.git

# Navigate to the project
cd Tic-Tac-Toe-Game

# Install dependencies
npm install

# Start the dev server
npm run dev
```

## 📁 Project Structure

```
src/
├── components/
│   ├── Board.jsx         # 3x3 grid layout
│   ├── Cell.jsx          # Individual cell with pop-in animation
│   ├── Confetti.jsx      # Winner celebration particles
│   ├── Scoreboard.jsx    # X / O / Draw score cards
│   └── StatusBar.jsx     # Turn indicator & result display
├── App.jsx               # Game logic & state management
├── App.css               # Component styles & animations
├── index.css             # Design tokens & global reset
└── main.jsx              # React entry point
```
